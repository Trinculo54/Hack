This is a simple package tweak that adds over 50 day and night wallpapers to the stock iOS 13 dark mode (appearance aware) wallpapers. Some are OLED friendly and are in Full HD.

There’s a script included in this tweak that you can find in /Documents titled as DarkPapersScript. Inside the script folder you’ll find a tutorial on how to manually add your own day and night wallpapers to this package, in case you want to add your own custom wallpapers.

This tweak packages depends on WallpaperLoader in order to work. You can find the tweak in Skittyblock’s repository at:

https://skitty.xyz

All wallpapers included in this package are free to use. I do not own any rights above them and the credit goes to the original creators. The only changes made to these wallpapers by me were to tweak the darkness/color in some of them and to modify their original size.
