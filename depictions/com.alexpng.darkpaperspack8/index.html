This is an essential pack for the tweak DarkPapers. It can be installed without DarkPapers in case you don't want the all-in-one package.

This pack includes the wallpapers: Licancabur 1 and Licancabur 2. Check the screenshots to see.

DarkPapers is a tweak that adds over 50 OLED friendly and Full HD day/night wallpapers (and with more coming in future updates) into the iOS 13 native dark mode wallpaper cycle (apperance aware). No setting up is required, simply install the tweak and go to the wallpaper section in the settings app and look in the still wallpapers to see your new dark mode compatible wallpapers.

All wallpapers included in this package are free to use. I do not own any rights above them and the credit goes to the original creators. The only changes made to these wallpapers by me were to tweak the darkness/color in some of them and to modify their original size.

This tweak is compatible with iOS 13 and above. The wallpapers included in the package are NOT optimized for iPads, only for iPhones. iPad users can (optionally) download the WallpaperLoader (Prefs) tweak, which is a different tweak from WallpaperLoader, in order to use their own custom wallpapers, or wait until the next update that adds iPad compatibility.

Special thanks to Skittyblock for the guide on how to add custom wallpapers into WallpaperLoader.

This package depends on WallpaperLoader in order to work. You can find the tweak in Skittyblock’s repository at:
